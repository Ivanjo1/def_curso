# Super Bowls Legendarios

Este proyecto es una página web sencilla que presenta una recopilación de los Super Bowls más recordados en la historia de la NFL, destacando los momentos más importantes de cada uno.

## Estructura del proyecto

- `index.html`: Página principal con información de los Super Bowls legendarios y una encuesta interactiva.
- `styles.css`: Estilos para la página web.
- `imagenes/`: Carpeta para las imágenes utilizadas en la página.
- `.vscode/settings.json`: Configuración recomendada para Visual Studio Code.

## Características

- Listado de los Super Bowls más emocionantes con descripciones e imágenes.
- Navegación interna para acceder rápidamente a cada Super Bowl.
- Formulario para que los visitantes voten por su Super Bowl favorito.
- Mensaje de agradecimiento tras enviar la encuesta.

## Uso

1. Clona o descarga este repositorio.
2. Asegúrate de tener las imágenes necesarias en la carpeta `imagenes/`.
3. Abre `index.html` en tu navegador para ver la página.
